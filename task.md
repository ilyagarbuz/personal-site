### Роль

Ты — senior full-stack разработчик, специализирующийся на Astro и TypeScript. Твоя задача — создать статический сайт-портфолио/блог для full-stack разработчика, следуя принципам **минимализма**, **производительности** и **доступности (a11y)**.

### Стек технологий

- **SSG:** Astro (latest version)
- **Язык:** TypeScript
- **Стили:** Чистый CSS (без Tailwind, SCSS или других препроцессоров)
- **Иконки:** `lucide` (использовать только через импорт компонентов)
- **Шрифты:** Локальные, путь: `/src/assets/fonts` (использовать `@font-face`)

### Основные архитектурные требования

1.  **Адаптивность:** Сайт должен корректно отображаться на всех устройствах. Различия между десктопной и мобильной версткой минимальны (только отступы и размер шрифтов).
2.  **Производительность:** Минимум клиентского JavaScript. Все интерактивные элементы должны работать без тяжелых скриптов. Использовать Astro Islands только при крайней необходимости.
3.  **Семантика и доступность:** Использовать семантические теги (`<header>`, `<main>`, `<section>`, `<article>`, `<nav>`, `<ul>`). Обеспечить контрастность цветов, `focus-visible` для интерактивных элементов и атрибуты `aria-label` для иконок без текста.
4.  **Дизайн-система:**
    - Преобладают **скругленные углы** (`border-radius`).
    - Основной контейнер контента: `max-width: 850px`, ширина на десктопе фиксированная.
    - **Темы:** Тёмная и светлая. Переключение через CSS-переменные (`data-theme` или класс на `html`). Плавный переход цветов (`transition: background-color 0.2s, border-color 0.2s`).
    - **Фон:** Ненавязчивый градиент или микро-текстура, который меняется в зависимости от темы (например, `radial-gradient(circle at 10% 20%, ...)`).
5.  **UI компоненты:** Кодовая база должна быть модульной. Общие элементы (Header, Footer, Card, Button, ThemeToggle) вынести в компоненты (`src/components`).

### Структура и функциональность страниц

#### 1. Шапка (Header)

- **Левый край:** Инициалы "IG". Шрифт: ultra-bold.
- **Правый край:** Иконки (GitHub, LinkedIn, Mail) + разделитель (пайп) + иконка переключения темы.
- **Поведение:** Sticky.
  - _Обычное состояние:_ Полная ширина, фон прозрачный/основной фон сайта.
  - _При скролле (sticky active):_ Ширина сужается до `fit-content` (или `max-width: 500px`), фон становится непрозрачным (слегка размытый или с затемнением), появляется внешняя обводка с `border-radius: 100px` (форма капсулы), добавляется box-shadow. Переход должен быть плавным (transition).
- **Адаптивность:** практически не чем не отличается от десктом версии, уменьщается лого и по высоте становится меньше, по ширене шапка резиновая

#### 2. Подвал (Footer)

- **Левый край:** "© 2026 Ilya Garbuz" (имя уточнить, т.к. в Hero другое).
- **Правый край:** Иконки (GitHub, LinkedIn, Mail). Без дублирования текста.
